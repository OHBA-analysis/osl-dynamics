Schaefer100 Parcellation
========================

This parcellation is a volumetric version of the Schaefer 100-parcel cortical parcellation from `Schaefer et al. (2018) <https://doi.org/10.1093/cercor/bhx179>`_. The parcels are derived from resting-state fMRI functional connectivity gradients.

Example Code
------------

Example code for plotting with this parcellation:

.. code::

    from osl_dynamics.analysis import power

    power.save(
        ...,
        mask_file="MNI152_T1_8mm_brain.nii.gz",
        parcellation_file="atlas-Schaefer_nparc-100_space-MNI_res-8x8x8.nii.gz",
        filename="map_.png",
    )
